## Deploy ke GitHub Pages (5 menit)

### Step 1 — Buat Repo Baru di GitHub

1. Buka [github.com/new](https://github.com/new)
2. Nama repo: `haiiii-alyaa`
3. Set ke **Public**
4. Klik **Create repository**

### Step 2 — Upload File-file Ini

```bash
# Clone / init repo lokal
git init
git remote add origin https://github.com/USERNAME/haiiii-alyaa.git

# Masukkin semua file dari folder ini
git add .
git commit -m
git push -u origin main
```

### Step 3 — Aktifkan GitHub Pages

1. Buka repo → **Settings** → **Pages**
2. Source: pilih **GitHub Actions**
3. Tunggu ~60 detik
4. Website live di: `https://USERNAME.github.io/haiiii-alyaa`

### Step 4 — Done! 🎉

Send the link. Tunggu reaksinya.
